## Document Processing

- [Docling](https://docling-project.github.io/docling/) — lib IBM pour extraire et convertir des documents (PDF, Word, PowerPoint...) en formats structurés (Markdown, JSON...)
- [Docling Vision](https://docling-project.github.io/docling/usage/vision_models/) — extension de Docling pour analyser les éléments visuels (figures, tableaux, schémas) via des modèles vision
- [Docling + Code Engine + GPU](https://github.com/IBM/CodeEngine/blob/main/serverless-fleets/tutorials/docling/README.md) — tutoriel pour exécuter Docling sur IBM Code Engine avec accélération GPU

## LLMs sur GPU

- [watsonx.ai](https://www.ibm.com/watsonx) — plateforme IBM pour entraîner, déployer et interroger des LLMs, avec accès GPU

## Modèles Granite

- [Granite Vision — Hugging Face](https://huggingface.co/collections/ibm-granite/granite-vision) — collection de modèles multimodaux IBM pour la compréhension d'images et de documents
- [Granite TimeSeries — Hugging Face](https://huggingface.co/collections/ibm-granite/granite-time-series) — collection de modèles IBM pour la prévision et l'analyse de séries temporelles
- [Granite — Ollama](https://ollama.com/search?q=granite) — modèles Granite disponibles pour exécution locale via Ollama
- [Granite TimeSeries Cookbook](https://github.com/ibm-granite-community/granite-timeseries-workshop) — notebooks et exemples d'utilisation des modèles Granite TimeSeries
